Pullable Data Fields
4 fields available
ℹ Per the SmartOffice contract, only these 4 fields are returned by the Pull API. Toggle which to import into your payroll system.
UserId   Employee Code
Maps to your StaffCode — used to identify the employee
LogDate   Punch Timestamp
ISO 8601 format — e.g. 2025-05-10T10:00:00
SerialNumber   Device Serial
Identifies which device captured the punch
DeviceSName   Device Display Name
Human-readable name e.g. "ME(Attendance)"
